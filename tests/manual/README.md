- **reduce.test.js** - Tests for the reduce() function
- **ceil.test.js** - Tests for the ceil() function

- **defaultTo.test.js** - Tests for the defaultTo() function

## Approach

These tests were written manually by analyzing the function documentation and source code to:
- Understand the expected behavior
- Identify edge cases
- Design comprehensive test scenarios
- Achieve high code coverage

This demonstrates traditional test-driven development practices without AI assistance.
